Company Program

How to Use

Welcome to the Company Program! Follow these steps to use the application:

  1. Enter your current salary in the "Salary" field. Only numbers are allowed.
  2. Specify the number of months worked (maximum 12) in the "Months" field.
  3. Choose an option from the menu:
    • New Salary: Calculates a new salary based on the provided rules.
    • Vacation: Calculates the value of vacation based on the current and new salary.
    • Thirteenth: Calculates the thirteenth salary based on the current and new salary for the specified months.
    • Logout: Logs you out of the application.
  4. Review the results displayed below the input fields.
  5. You can click the "Reset Fields" button to clear the input fields and results.
